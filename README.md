# 🚀 Nexus API – Backend Developer Assignment

## 📌 Overview

This project is a backend API built as part of a developer assignment. It provides structured endpoints to handle data operations efficiently using modern web technologies.

---

## 🛠️ Tech Stack

* **Node.js**
* **TypeScript**
* **Vite**
* **REST API Architecture**

---

## 📂 Project Structure

```
├── components/        # Reusable components
├── src/               # Main source code
├── lib/               # Utility functions
├── server.ts          # Entry point of backend server
├── db.ts              # Database configuration
├── index.html         # Frontend entry (if applicable)
├── package.json       # Project dependencies
├── tsconfig.json      # TypeScript configuration
├── vite.config.ts     # Vite configuration
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Adi8454/Nexus-API.git
cd Nexus-API
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the server

```bash
npm run dev
```
